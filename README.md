# GT Prax

The web app helps with organization in your practice.

## Frontend

 * Razor Pages in ASP.NET Core
 * Bulma Framework
 * jQuery 
 * Font Awesome
 
## Backend

* .NET Core 10.0
  * ASP.NET Core
* Template Engine Scriban
* Logging via Serilog
* SQLite Persistence
